# ML Regression Practice

A hands-on regression project built with Python and scikit-learn,
practicing core ML concepts from Linear to Logistic Regression.

## Topics Covered
- Linear Regression — predict exam score from study hours
- Multiple Regression — multiple features to one output
- Polynomial Regression — curved relationships in data
- Logistic Regression — binary classification (churn prediction)

## Tools & Libraries
- Python
- scikit-learn
- NumPy
- Google Colab

## Project Structure
linear_regression.ipynb — study hours vs exam score model
